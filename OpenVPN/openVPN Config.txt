[6] CONNECTIVITY VALIDATION STEPS

After connecting from the Windows client:

1. Verify VPN address on the client:
   - Run `ipconfig` and confirm an "OpenVPN TAP-Windows6" adapter
     with an IP in 10.8.0.0/24 (e.g., 10.8.0.2).

2. Test reachability of the DMZ web server (through VPN):
   - `ping 192.168.2.10`
   - Browse to `http://192.168.2.10` and confirm the Apache default page.
